# generate-tech-proposal

根据图片、文字描述或蓝湖链接生成项目技术方案的 Skill。

## 功能

- **图片输入** - 分析 UI 设计、流程图、架构图，提取功能需求
- **文字描述** - 解析需求文档，识别核心模块和业务规则
- **蓝湖链接** - 获取 Figma 设计稿数据，分析页面结构和交互逻辑
- **复用检测** - 检查已有功能，标注复用/改造/新建

## 输出

生成包含以下内容的完整技术方案：

1. 需求背景
2. 项目原型
3. 本期目标
4. 影响范围评估
5. 业务分析（用例、流程、状态图）
6. 系统设计（时序图、数据模型、缓存、消息）
7. 接口设计
8. 非功能性设计

## 使用方式

在 Agent 对话中提供以下任一输入：
- 需求图片
- 功能描述文字
- 蓝湖/Figma 设计稿链接

Agent 将自动调用此 Skill 生成技术方案文档。
